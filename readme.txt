== Description ==

A small live chat module that allows members/guests to chat with each other as they browse your site. 
Includes public chat as well as private messaging. 
Discussions carry across pages as users browse your site.

Uses: 
Jquery & Jquery UI
SQL Database

For more information visit and help visit: http://alexmakeswebsites.com/integrate-ajax-chat-into-your-site-using-phpmysql/


== Installation ==

1. Upload `amw-chat` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
